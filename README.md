## Flagship Project

### [mini-linux](https://github.com/livaiyena/mini-linux) — Complete Embedded Linux System

A **production-grade, from-scratch embedded Linux telemetry system** targeting ARM64 (AArch64), demonstrating advanced systems engineering across the entire stack.

**What's Included:**
- **Custom Linux Kernel Module (LKM)** — I2C character device driver with file operations and kernel memory management
- **IPC Architecture** — POSIX shared memory with concurrent reader/logger applications ensuring data integrity
- **Cross-Compilation Pipeline** — Docker-containerized ARM64 toolchains (binutils, gcc, musl) for reproducible builds
- **Minimal RootFS** — Custom BusyBox-based Linux filesystem hierarchy from scratch
- **QEMU Simulation** — Full ARM64 system emulation with automated verification tests
- **Web Dashboard** — Real-time telemetry visualization via BusyBox httpd with CGI scripting
- **AI-Powered Diagnostics** — Integrated Google Gemini API for automated log analysis
- **CI/CD Pipeline** — GitHub Actions automation for cross-compilation and smoke testing

**Technologies:** C · Linux Kernel · ARM64 · POSIX APIs · Docker · QEMU · Bash · Git

**Key Achievement:** End-to-end embedded system from kernel driver → userspace applications → filesystem generation → automated testing.
